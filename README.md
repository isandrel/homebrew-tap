# Isandrel Tap

## How do I install these formulae?

`brew install isandrel/tap/<formula>`

Or `brew tap isandrel/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "isandrel/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
